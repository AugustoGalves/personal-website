Engenharia Web do Zero

Para estabelecer minha presença digital de forma sólida, decidi projetar e desenvolver este site portfólio integralmente do zero. O projeto foi executado entre dezembro de 2025 e abril de 2026, com mais de 200 commits. O foco foi aplicar engenharia de software no front-end: arquitetura de informação limpa, performance de elite e domínio total sobre a infraestrutura da aplicação.

O Desafio Técnico

A decisão mais crítica foi abrir mão de frameworks modernos (como React, Vue ou Tailwind). O desafio era construir um sistema modular, escalável e extremamente rápido, utilizando exclusivamente as tecnologias nativas da web.

Arquitetura e Implementação

Desenvolvi uma solução robusta focada nos fundamentos, abrangendo desde a criação de componentes lógicos até o gerenciamento de registros DNS, garantindo uma presença digital profissional e segura.

  • Web Components Customizados: Criação de componentes JS reutilizáveis (como headers, cards e footers) nativamente, reduzindo o volume de código repetido sem depender do React.
  • Design de UI/UX Exclusivo: Interface concebida do zero, enriquecida com a criação de uma biblioteca de ícones pessoais desenhada sob medida para a marca.
  • SEO & Performance: Otimização semântica, carregamento assíncrono de scripts, minificação de recursos e integração completa de Structured Data (JSON-LD) e Open Graph Tags.
  • Infraestrutura & Redes: Deploy contínuo via GitHub Pages acoplado ao Cloudflare para gerenciamento de DNS, regras de cache de CDN e configuração de e-mail profissional via rotas de redirecionamento.

Resultados

O resultado é uma plataforma de marca pessoal de altíssimo desempenho tecnológico. A ausência de dependências de terceiros garante que o site nunca ficará obsoleto por atualizações de frameworks.

  • Tempo de carregamento e First Contentful Paint (FCP) próximos da perfeição no Lighthouse.
  • Apresentação técnica que comprova fluência em programação e infraestrutura, indo além de conhecimentos superficiais.
  • Ambiente totalmente escalável para adição de novos projetos de engenharia e software com facilidade.
